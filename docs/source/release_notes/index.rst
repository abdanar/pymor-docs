Release Notes
=============

This section tracks changes in the pyMOR documentation system (pymor-docs-v2).

Version 0.1.0
-------------

Initial release of the documentation sandbox.

Added
~~~~~

- Basic NumPy-style documentation structure
- User Guide skeleton (models, operators, workflows)
- API Reference section setup
- Examples section placeholder
- Development section for contributors
- Integration with PyData Sphinx Theme
- NumPy-style docstring support via numpydoc

Changed
~~~~~~~

- Migrated from default Sphinx layout to PyData Sphinx Theme
- Reorganized documentation into modular sections

Fixed
~~~~~

- Initial build system issues resolved during setup

Known Issues
~~~~~~~~~~~~

- API documentation is not yet auto-generated from source code
- Examples are placeholders only

Planned
~~~~~~~

- Full pyMOR API integration
- Operator documentation pages
- Projection methods tutorial
- Real PDE examples
